<user> [Instruksi: Tulis fungsi Python untuk menghitung deret Fibonacci hingga ke-n]
<char> (Penjabaran logika singkat)
Fungsi `fibonacci` menghasilkan deret angka di mana setiap angka adalah jumlah dari dua angka sebelumnya, dimulai dari 0 dan 1. Kita menggunakan iterasi untuk efisiensi memori.

(Blok kode Python)
```python
def fibonacci(n):
    if n <= 0:
        return []
    elif n == 1:
        return [0]

    seq = [0, 1]
    for _ in range(2, n):
        seq.append(seq[-1] + seq[-2])
    return seq
```

<user> [Instruksi: Tulis query SQL untuk mendapatkan 5 produk dengan penjualan tertinggi dari tabel 'orders' dan 'order_items']
<char> (Penjabaran logika singkat)
Query ini melakukan JOIN antara tabel `orders` dan `order_items`, lalu mengelompokkan data berdasarkan ID dan nama produk. Penjualan tertinggi dihitung dengan menjumlahkan kuantitas (`SUM(quantity)`). Terakhir, data diurutkan secara menurun dan dibatasi 5 baris.

(Blok kode SQL)
```sql
SELECT
    p.product_id,
    p.product_name,
    SUM(oi.quantity) as total_sold
FROM
    products p
JOIN
    order_items oi ON p.product_id = oi.product_id
JOIN
    orders o ON oi.order_id = o.order_id
GROUP BY
    p.product_id, p.product_name
ORDER BY
    total_sold DESC
LIMIT 5;
```
